# Bokii Cool Solutions

Marketing website for Bokii Cool Solutions, a mobile fridge, aircon, and refrigeration repair business.

## Includes

- Responsive landing page
- Branded header and hero section
- Service overview
- Contact actions for phone and WhatsApp
- Project logo asset

## Project Files

- `index.html`
- `styles.css`
- `assets/logo-main.png`

## Notes

The site is currently a static front-end build and can be deployed to GitHub Pages or any static hosting platform.
